# SIMC Study Tool

A mobile-friendly flashcard prototype for reviewing SIMC intro and intermediate lessons.

## Open Locally

Open `index.html` in a browser.

## Files

- `index.html` - app page
- `styles.css` - visual styling
- `app.js` - flashcard interactions
- `data.js` - original lesson data
- `data-extra.js` - imported SIMC lesson cards
